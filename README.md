# Spark -> OpenAI API wrapper

This server wraps Spark's API as OpenAI's

## Install

First, install `pdm`

```shell
pip install pdm
```

Sencond, setup this project

```shell
pdm install
```

## Deploy

You need to set 3 environment variables:

```env
APPID=
APISecret=
APIKey=
```

Start the development server:

```shell
pdm dev
```
